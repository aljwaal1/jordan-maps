# خرائط الأردن أونلاين

تطبيق Flutter Android لعرض خريطة الأردن أونلاين مع بحث داخل الأردن، بحث تجاري حول موقعك، رسم مسار قيادة، ووضع قيادة مبسط يعطي تعليمات يمين ويسار ودوار ومتابعة الموقع.

## المزايا

- خريطة الأردن أونلاين عبر OpenStreetMap.
- البحث عن أسماء الأماكن داخل الأردن فقط.
- بحث فئات تجارية حول موقعك أو مركز الخريطة: محلات، بقالة، مطاعم، كافيه، صيدليات، وقود، بنوك، مستشفيات، مدارس.
- تحديد موقع المستخدم GPS.
- رسم مسار قيادة من موقعك إلى الوجهة عبر OSRM.
- وضع قيادة: سهم واتجاه المناورة القادمة، يمين/يسار/دوار/تابع للأمام، متابعة الموقع أثناء الحركة.
- تنبيه عند الابتعاد عن المسار مع زر إعادة حساب.
- واجهة عربية RTL داكنة ومناسبة للشاشات الصغيرة.
- GitHub Actions جاهز لإخراج APK debug.

## ملاحظات مهمة

- التطبيق أونلاين: الخريطة والبحث والمسارات تحتاج إنترنت.
- بيانات البحث والمسارات تعتمد على OpenStreetMap، لذلك دقة المحلات والمسارات حسب توفر البيانات في الأردن.
- المسارب التفصيلية مثل Google Maps غير مضمونة، لكن التطبيق يعطي توجيهات يمين ويسار ودوارات وخط المسار.
