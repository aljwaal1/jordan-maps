# خرائط الأردن أونلاين

تطبيق Flutter Android يعرض خريطة الأردن أونلاين مع بحث داخل الأردن، تحديد موقعي، واختيار نتيجة البحث ورسم مسار قيادة من الموقع الحالي إلى الوجهة.

## المزايا الحالية

- خريطة OpenStreetMap أونلاين.
- بحث داخل الأردن فقط عبر Nominatim.
- تحديد موقع المستخدم GPS.
- عرض نتيجة البحث على الخريطة.
- رسم مسار قيادة عبر OSRM.
- عرض المسافة والزمن التقريبي.
- نسخ إحداثيات الوجهة.
- واجهة عربية RTL وتصميم داكن.

## البناء عبر GitHub Actions

1. ارفع هذا المشروع إلى GitHub.
2. ادخل إلى Actions.
3. شغل Workflow باسم: Build Jordan Maps APK.
4. بعد انتهاء البناء، نزّل ملف APK من Artifacts.

## ملاحظات مهمة

- التطبيق يحتاج إنترنت لعرض الخريطة والبحث وحساب المسار.
- التطبيق يحتاج إذن الموقع لتحديد موقعي.
- البحث محصور بالأردن عبر `countrycodes=jo`.
- هذه نسخة بداية، ويمكن لاحقًا إضافة تسجيل الطريق والرجوع من نفس الطريق.
